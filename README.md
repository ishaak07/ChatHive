# 🐝 ChatHive

A full-stack real-time chat application built with the **MERN Stack** and **Socket.IO**. 
ChatHive enables users to connect through private and group conversations while offering AI-powered productivity features and an interactive multiplayer gaming experience.

## 🚀 Live Demo

* **Frontend:** https://chat-hive-red.vercel.app/
* **Backend:** https://chathive-backend-kbxa.onrender.com

## ✨ Features

### 🔐 Authentication

* Secure user registration and login
* JWT-based authentication and authorization
* Protected routes

### 💬 Real-Time Communication

* One-to-one private messaging
* Real-time group chats
* Instant message delivery using Socket.IO
* Typing indicators
* Online/offline user status
* Delete messages
* Emoji picker support

### 👥 Friend Management

* Search users
* Send friend requests
* Accept or reject requests
* Remove friends

### 🏠 Chat Rooms

* Create chat rooms
* Join existing rooms
* Leave rooms
* Delete rooms (where permitted)

### 👤 User Profiles

* Profile avatar customization
* About/Bio section
* Interest tags

### 🤖 AI-Powered Features

* **Smart Reply** – Generates contextual reply suggestions.
* **Chat Summarizer** – Creates concise summaries of conversations.
* **Smart Extractor** – Extracts important information from chat messages.
* Optimized with faster responses and timeout handling.

### 🎮 Multiplayer Gaming

* Real-time multiplayer Connect 4 game
* Live gameplay synchronized with Socket.IO

### 📱 User Experience

* Fully responsive design
* Modern and intuitive interface
* Fast real-time updates

## 🛠️ Tech Stack

**Frontend**

* React.js
* Vite
* Axios
* Socket.IO Client

**Backend**

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT
* bcrypt.js
* Socket.IO

**Deployment**

* Frontend: Vercel
* Backend: Render
* Database: MongoDB Atlas
